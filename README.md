# 🔐 Login Authentication System

A simple *Login & Registration Authentication System* built using *Node.js, Express, JWT, and bcrypt*.
This project demonstrates secure user authentication with password hashing and protected routes.

---

## 🚀 Live Demo

🔗 https://oibsip-login-auth.onrender.com

---

## 📂 GitHub Repository

🔗 https://github.com/maitrishivhare/OIBSIP-login-auth

---

## ✨ Features

* User Registration
* Secure Login Authentication
* Password Hashing using *bcrypt*
* Token-based authentication using *JWT*
* Protected Dashboard Route
* Logout functionality
* Simple and responsive UI

## ⚙️ Installation & Setup

1️⃣ Clone the repository


git clone https://github.com/maitrishivhare/OIBSIP-login-auth.git


2️⃣ Navigate to project folder


cd OIBSIP-login-auth/login-auth


3️⃣ Install dependencies


npm install


4️⃣ Run the server


node server.js


5️⃣ Open in browser


http://localhost:3000
